HTML Lists

HTML provides two main types of lists:

  • Unordered lists use the <ul> tag and display items with bullet points
  • Ordered lists use the <ol> tag and display items with numbers

Each item within either type of list must be wrapped in <li> (list item) tags.

Example

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>
